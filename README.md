# パワポ画像貼り付けツール

PowerPointで作成したフォーマット位置に画像を出力するGUIツールです。

## 概要

PowerPointで作成したフォーマットを使用して画像貼り付け・テキスト置換を行います。
フォーマットは「特定レイアウト」と「順番に貼り付け」に対応しています。

- 四角形 → 画像 への置換
  - 四角形の配置 = 画像の配置
  - 四角形のテキスト = 画像のラベル
- テキストの置換
  - @1 (@ + 数字) = グループ名への置換
  - #1 (# + 数字) = ラベル名への置換

## 動作環境

アプリケーションを動作させるために以下の環境が必要です。

- Python 3.10.2
- python-pptx 0.6.21
- PySimpleGUI 4.59.0
